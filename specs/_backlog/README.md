# Feature Backlog

This directory stores deferred feature ideas that are not yet promoted to an active numbered spec.

When promoting a backlog item:
1. Create a new numbered folder in `specs/` (for example `specs/002-some-feature/`).
2. Move and expand the item into `spec.md`, then create `plan.md` and `tasks.md`.
3. Remove or link back from the backlog entry.
