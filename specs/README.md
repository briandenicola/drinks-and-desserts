# Specs Structure

Active feature work is organized under numbered folders:

- `specs/NNN-feature-name/spec.md`
- `specs/NNN-feature-name/plan.md`
- `specs/NNN-feature-name/tasks.md`

Supporting artifacts for a feature (for example `research.md`, `data-model.md`, `quickstart.md`) live alongside that feature's core files.

Future ideas and deferred work are tracked in `specs/_backlog/`.
